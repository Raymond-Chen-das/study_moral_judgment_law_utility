## 1. 古希臘哲學家

### 柏拉圖 (Plato, 約427-347 BC)

**📍 基本資料**

- 時代: 古希臘(公元前4世紀)
- 學派: 理型論(Theory of Forms)
- 代表作: 《理想國》(The Republic)

**💡 核心理論**

- **洞穴寓言(Allegory of the Cave)**
  - 人們如同洞穴囚徒,只能看見牆上的影子
  - 誤以為影子就是真實世界
  - 哲學家的任務是走出洞穴,看見真實的陽光
  
**🎯 在第6章的應用**

- 質疑數據是「真實的道德判斷」還是「牆上的影子」
- 提醒我們:參與者的滑鼠點擊可能只是行為,不是真正的道德選擇
- 核心警示:**數據可能只是現實的投影,而非現實本身**

**🔑 記憶要點**

```
柏拉圖 = 洞穴寓言 = 影子vs.真實
→ 應用: 數據是影子還是真相?
→ 教訓: 保持謙遜,質疑表象
```

---

### 亞里斯多德 (Aristotle, 384-322 BC)

**📍 基本資料**

- 時代: 古希臘(柏拉圖的學生)
- 學派: 德行倫理學(Virtue Ethics)
- 代表作: 《尼各馬科倫理學》(Nicomachean Ethics)

**💡 核心理論**

- **實踐智慧(Phronesis)**
  - 道德不是遵守固定規則
  - 而是在具體情境中做出恰當判斷的能力
  - 需要經驗、判斷力和情境敏感性
  
- **中庸之道(Golden Mean)**
  - 德行在兩個極端之間
  - 例如勇敢介於魯莽和懦弱之間

**🎯 在第6章的應用**

- 呼應「情境主義」的發現
- 人類的道德判斷本質上是情境依賴的
- 核心洞察:**道德判斷需要在情境中運用智慧**

**🔑 記憶要點**

```
亞里斯多德 = 實踐智慧 = 情境判斷
→ 應用: 支持情境主義
→ 教訓: 道德不是套公式,而是智慧運用
```

## 2. 啟蒙時代哲學家

### 休謨 (David Hume, 1711-1776)

**📍 基本資料**

- 時代: 18世紀蘇格蘭啟蒙運動
- 學派: 經驗主義(Empiricism)
- 代表作: 《人性論》(A Treatise of Human Nature)

**💡 核心理論**

- **休謨的鍘刀(Hume's Guillotine) / 是-應該問題(Is-Ought Problem)**
  - 不能從「是什麼」(事實陳述)直接推導出「應該怎樣」(價值判斷)
  - 例如:
    - ✗ 「人們選擇效益主義」(is) → 「所以效益主義是對的」(ought)
    - ✓ 需要額外的道德論證來連接事實與價值

**🎯 在第6章的應用**

- 警告不要把描述性發現直接當成規範性結論
- 「77%選效益主義」是事實,不代表「效益主義就是對的」
- 核心提醒:**從數據到道德判斷,需要獨立的倫理論證**

**🔑 記憶要點**

```
休謨 = 是-應該問題 = 事實≠價值
→ 應用: 統計發現≠道德準則
→ 教訓: 描述性≠規範性,需要額外論證
```

---

### 康德 (Immanuel Kant, 1724-1804)

**📍 基本資料**

- 時代: 18世紀德國啟蒙運動
- 學派: 義務論(Deontology)
- 代表作: 《道德形上學基礎》(Groundwork for the Metaphysics of Morals)

**💡 核心理論**

- **理性的自由意志**
  - 道德判斷必須出於理性,而非衝動或偶然
  - 道德行為源於對義務的尊重,不是因為後果
  
- **絕對命令(Categorical Imperative)**
  - 「要如此行動,使你的行為準則能成為普遍法則」
  - 強調道德的普遍性和必然性

**🎯 在第6章的應用**

- 質疑大部分參與者的選擇可能只是「隨機點擊」
- 如果不是出於理性思考,算道德判斷嗎?
- 核心挑戰:**真正的道德選擇需要理性自由意志**

**🔑 記憶要點**

```
康德 = 理性自由意志 = 義務論
→ 應用: 質疑「隨機點擊」是否算道德選擇
→ 教訓: 道德需要理性,不是衝動
```

## 3. 19世紀哲學家

### 彌爾 (John Stuart Mill, 1806-1873)

**📍 基本資料**

- 時代: 19世紀英國維多利亞時代
- 學派: 效益主義(Utilitarianism)
- 代表作: 《效益主義》(Utilitarianism)

**💡 核心理論**

- **最大多數人的最大幸福(Greatest Happiness Principle)**
  - 道德的標準是行為是否能帶來最大幸福
  - 正確的行為是能產生最多快樂、最少痛苦的行為
  
- **質與量的效益**
  - 區分「高級快樂」(精神、智識)和「低級快樂」(肉體、感官)
  - 「當一個不滿足的蘇格拉底,勝過當一個滿足的豬」

**🎯 在第6章的應用**

- 77%的人選擇「救多數人」似乎支持效益主義
- 但這只是表象,不代表效益主義就是「對的」
- 核心問題:**多數人的選擇≠道德真理**

**🔑 記憶要點**

```
彌爾 = 效益主義 = 最大幸福
→ 應用: 77%選效益主義,但這不代表它「對」
→ 教訓: 描述性統計≠規範性結論
```

## 4. 20世紀哲學家

### 華特·米歇爾 (Walter Mischel, 1930-2018)

**📍 基本資料**

- 時代: 20世紀心理學家
- 學派: 社會認知心理學
- 代表研究: 棉花糖實驗(Marshmallow Test)

**💡 核心理論**

- **情境主義(Situationism)**
  - 人的行為主要由情境決定,而非穩定的內在特質
  - 挑戰傳統的「人格特質論」(Trait Theory)
  - 同一個人在不同情境中會有不同行為

**🎯 在第6章的應用**

- 研究發現強力支持米歇爾的情境主義
- 53.4%的預測力來自情境因素
- 僅25.2%來自個人特質
- 核心發現:**情境 > 特質**

**🔑 記憶要點**

```
米歇爾 = 情境主義 = 行為受情境驅動
→ 應用: SHAP分析顯示情境因素53.4%
→ 教訓: 人不是固定的,會隨情境改變
```

---

### 沙特 (Jean-Paul Sartre, 1905-1980)

**📍 基本資料**

- 時代: 20世紀法國存在主義
- 學派: 存在主義(Existentialism)
- 代表作: 《存在與虛無》(Being and Nothingness)

**💡 核心理論**

- **存在先於本質(Existence Precedes Essence)**
  - 人沒有預設的本質,自己創造自己的本質
  
- **自由與責任**
  - 人在任何情境下都有選擇的自由
  - 這種自由意味著完全的責任
  - 不能把責任推給情境、社會或命運
  
- **「人被判定是自由的」**
  - 自由是負擔,因為必須為選擇負責

**🎯 在第6章的應用**

- 挑戰情境主義的道德意涵
- 如果選擇受情境驅動,我們還需要負責嗎?
- 沙特會說:即使情境影響很大,你仍有選擇的自由
- 核心張力:**情境影響 vs. 個人責任**

**🔑 記憶要點**

```
沙特 = 存在主義 = 自由與責任
→ 應用: 質疑「情境決定論」是否免除責任
→ 教訓: 情境影響≠沒有責任
```

---

### 海德格 (Martin Heidegger, 1889-1976)

**📍 基本資料**

- 時代: 20世紀德國哲學家
- 學派: 現象學(Phenomenology)、存在主義
- 代表作: 《存在與時間》(Being and Time)

**💡 核心理論**

- **技術的危險(The Danger of Technology)**
  - 現代技術不只是工具,而是一種世界觀
  - 技術思維把世界簡化為可計算、可控制的對象
  - 遮蔽了世界的豐富性和神秘性
  
- **「存在的遺忘」(Forgetfulness of Being)**
  - 過度關注「存在者」(beings)而忘記「存在」(Being)本身

**🎯 在第6章的應用**

- 警告不要過度依賴數據分析
- 「鐵鎚很有用,但如果你只有鐵鎚,你會把所有問題都看成釘子」
- 核心警示:**工具不是答案,數據不能量化一切**

**🔑 記憶要點**

```
海德格 = 技術的危險 = 工具理性的限制
→ 應用: 數據分析不能量化道德的全部
→ 教訓: 數字有用,但不要忘記無法量化的部分
```

## 5. 東方哲學

### 孔子 (Confucius, 551-479 BC)

**📍 基本資料**

- 時代: 中國春秋時期
- 學派: 儒家(Confucianism)
- 代表作: 《論語》(Analects)

**💡 核心理論**

- **「權」的智慧**
  - 「可與共學,未可與適道;可與適道,未可與立;可與立,未可與權」
  - 「權」= 權變、權衡,在情境中彈性運用原則的智慧
  - 不是沒有原則,而是原則要在具體情境中活用
  
- **仁(Ren) - 關係倫理**
  - 道德不是抽象普遍規則,而是在關係中體現
  - 「親親而仁民,仁民而愛物」

**🎯 在第6章的應用**

- 解釋台灣的獨特定位
- 台灣既不極端效益主義,也不極端守法
- 可能反映儒家的「權變」文化傳統
- 核心洞察:**情境彈性 vs. 固定原則**

**🔑 記憶要點**

```
孔子 = 儒家 = 權變智慧
→ 應用: 台灣的道德彈性(不極端效益、不極端守法)
→ 教訓: 原則在情境中活用
```

## 📖 核心哲學教訓總結

### 教訓一:從柏拉圖學會謙遜
>
> 數據是一扇窗,但不是全景窗。永遠要質疑:我們看到的是真相,還是影子?

### 教訓二:從休謨學會區分「是」與「應該」
>
> 描述性發現(人們怎麼選)不等於規範性結論(人們應該怎麼選)。從事實到價值,需要獨立的倫理論證。

### 教訓三:從亞里斯多德學會情境智慧
>
> 道德判斷不是套用公式,而是在具體情境中運用實踐智慧。情境主義某種程度驗證了這個古老洞見。

### 教訓四:從沙特學會責任
>
> 即使情境影響很大,我們仍然有選擇的自由,因此仍需為選擇負責。情境影響≠道德免責。

### 教訓五:從海德格學會工具的限制
>
> 數據分析是強大的工具,但工具不是答案。有些關於意義、責任、人性的問題,本來就不是數字能回答的。

### 時間軸記憶法


**關於「真相與表象」**

- 柏拉圖:洞穴寓言 → 數據可能只是影子

**關於「事實與價值」**

- 休謨:是-應該問題 → 描述≠規範
- 彌爾:效益主義 → 多數選擇≠道德真理

**關於「情境與特質」**

- 亞里斯多德:實踐智慧 → 情境中的判斷
- 米歇爾:情境主義 → 行為受情境驅動
- 孔子:權變 → 原則的彈性運用

**關於「自由與責任」**

- 康德:自由意志 → 理性的道德選擇
- 沙特:存在主義 → 自由意味著責任

**關於「技術與人性」**

- 海德格:技術的危險 → 工具不能量化一切
