# MIT Moral Machine 跨文化道德判斷分析

## 🎯 研究問題
**道德判斷究竟是由個人特質（Trait）還是情境因素（Situation）主導？** 本專案透過分析 MIT Moral Machine 大規模資料集（從 1,128 萬筆精煉至 18.7 萬筆有效樣本），探討全球在「守法 vs. 效益主義」衝突中的選擇模式，並系統性定位台灣在東亞與全球的道德座標。

---

## 📊 核心發現 (Core Findings)
1. **情境主義（Situationism）獲得強力支持**：約 86% 的道德選擇變異來自情境因素。國家文化影響僅約 0.25% (ICC = 0.0025)。
2. **全球偏好效益主義**：整體守法率僅 22.6%（行層級）。當拯救多數生命與遵守法律發生衝突時，多數人傾向於選擇效益極大化。
3. **文化差異統計顯著但影響微弱**：東方 (Eastern) 與西方 (Western) 文化圈在守法率上雖有 5% 的顯著差異 (p < .001)，但效果量 (Cramér's V) 僅為 0.032，屬於極小效應。
4. **台灣定位特殊**：台灣的效益主義偏好為 0.393（全球倒數第二，僅高於日本），守法偏好為 0.347（東亞四國最低）。在道德距離上，台灣與韓國最為相似。
5. **「道德人格」多為測量假象**：潛在類別分析 (LCA) 敏感度分析顯示，當使用者觀測次數增加時，極端人格比例從 100% 大幅降至 54.7%，證實初步觀察到的穩定人格主要源於資料稀疏 (Data Sparsity)。

## 🛠️ 統計方法與研究邏輯 (Methodology Recap)
本研究採用 **三角驗證（Triangulation）** 策略，確保跨方法的結論一致性。

| 分析層級 | 統計手法 | 使用目的 (Why we use it) | 關鍵指標與學術貢獻 |
| :--- | :--- | :--- | :--- |
| **資料處理** (Ch 2) | **特徵工程與數據清理** | 篩選「法律 vs. 效益」衝突場景，定義義務論與效益主義的直接對抗。 | 樣本數演變：1,128 萬 → 377,592 行 → **187,124 行 (93,562 場景)**。 |
| **描述探索** (Ch 3) | **AMCE 比較與分群分析** | 理解全球道德分佈與台灣的定位，嘗試識別道德集群。 | 證實台灣為「雙低」特徵；**誠實報告失敗**：Cophenetic 係數僅 0.386，顯示道德偏好為連續光譜而非離散群聚。 |
| | **潛在類別分析 (LCA)** | 驗證個人是否存在穩定的道德人格特質。 | **敏感度分析**揭示 97.7% 的極端人格為假象。隨觀測增加，極端性降至 54.7%，駁斥強烈特質論。 |
| **推論驗證** (Ch 4) | **卡方檢定 & 效果量** | 檢定文化圈對選擇的影響，區分統計顯著性與實質重要性。 | **Cramér's V = 0.032**。強調在大樣本下，p 值主要反映樣本量，效果量才反映影響力。 |
| | **邏輯迴歸 (Logistic Regression)** | 評估宗教、政治立場等個人因素在控制環境後的淨效應。 | **Pseudo R² = 0.0002**。反直覺發現：宗教虔誠與政治進步程度均與守法呈**負相關** (OR < 1.0)。 |
| | **階層線性模型 (HLM)** | 處理「個人巢狀於國家」結構，分解國家間與國家內的變異。 | **ICC = 0.25% (0.0025)**。證明 99.75% 變異來自國家內部，為**情境主義**提供核心統計支持。 |
| **預測整合** (Ch 5) | **XGBoost 與 SHAP 解釋** | 探索預測力天花板 (Upper Bound)，解構各層級變數的貢獻。 | **AUC = 0.638**。SHAP 權重：**情境特徵 (53.4%)** > 個人因素 (25.2%) > 國家文化 (19.4%)。 |
| | **跨層級整合模型** | 結合 HLM 隨機效應與 AMCE 數據，解釋微觀與宏觀的矛盾。 | 透過**大數法則**解釋：微觀上文化影響極微，但宏觀聚合後可觀察到穩定的文化基線。 |

## 📖 研究架構細節
- **第 2 章：資料處理**：從 11M 筆原始數據清理出針對「法律 vs. 效益」困境的 93,562 個有效場景。
- **第 3 章：探索性分析**：包含全球道德地圖繪製、台灣定位分析，以及關鍵的 **LCA 敏感度分析（揭示測量假象）**。
- **第 4 章：統計推論**：運用卡方檢定、邏輯迴歸，以及本研究核心模型 **HLM (ICC=0.25%)** 進行因果探討。
- **第 5 章：預測模型與整合**：使用 XGBoost 探索預測上限 (AUC=0.638)，並利用**大數法則**進行跨層級的理論整合。
- **第 6 章：哲學批判**：針對「情境主義」的實證結果進行哲學辯證，區分事實 (Is) 與規範 (Ought)。

---

## 🏆 主要貢獻
- **實證支持情境主義**：透過 130 國數據證實道德判斷受情境驅動（約 86%）而非固定文化特質。
- **方法論嚴謹性**：堅持「效果量優先」原則，並誠實報告 LCA 與分群分析中未達標準的失敗發現，提供更高信度的學術洞察。
- **台灣道德座標定位**：首次系統性呈現台灣與東亞鄰國及全球文化圈在道德偏好上的量化差異。

---

## ⚠️ 研究限制
- **資料特性**：存在自我選擇偏誤 (Self-selection bias)，且 91.9% 的使用者僅有 1 次決策記錄，限制了個人穩定性的分析。
- **方法限制**：作為觀察性研究，無法確立強因果關係；二維道德光譜雖利於理解，但可能簡化了真實世界的道德複雜度。
- **詮釋警語**：實驗室點擊行為不完全等同於現實道德決策，應避免過度詮釋為絕對的道德傾向。
